# Contributing to RupeeRadar

Thank you for your interest in improving RupeeRadar!

## Getting started

1. Fork the repository and clone your fork.
2. For the API, run `npm install` inside `project2-backend/`.
3. Open `project1-frontend/index.html` in a browser to test the UI.

## Pull requests

- Keep changes focused — one feature or fix per PR when possible.
- Match existing code style (plain HTML/CSS/JS on the frontend; CommonJS on the backend).
- Update `README.md` or `API.md` if you change setup steps or endpoints.
- Test locally before submitting.

## Reporting issues

Open a GitHub issue with:

- What you expected vs. what happened
- Steps to reproduce
- Browser or Node.js version (if relevant)

## Code of conduct

Be respectful and constructive. This project was built as part of the DecodeLabs Industrial Training program.
